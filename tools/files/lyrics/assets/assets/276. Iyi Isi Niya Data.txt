276. Iyi Isi Niya Data

1.
Iy' isi n' iya Data, Kand' iyo ntez' amatwi, 
Numv' ibiremwa bisingiza izina rye mu ndirimbo, 
Isi n' iya Data, Mpora mbitekereza, 
Iby' Uhoraho yaremye byose, ni byiza bitangaje. 

2.
Iy' isi n' iya Data, Inyoni ziririmba, 
Umuseke wa mu gitondo, N' impumuro y' uburabyo, 
Ibimera byose, Hamwe n' ibyo mw' ijuru, 
Byose bigaragaz' ububasha, N' imbaraga ze nyinshi. 

3.
Iy' isi n' iya Data, N' ubwo yuzuy' ibyaha, 
Nta kizaganz' imbaraga ze, Azahor' ari ku ngoma, 
Isi n' iya Data, Mbese nakwiganyira?
Imana yac' izahora ku ngom' ibihe bidashira.
 
4.
Iy' isi n' iya Data, Ntabwo nzabyibagirwa, 
N' ubw' ibyaha biyuzuyemo, Imana n' umuyobozi, 
Isi n' iya Data, Ni kuki nababara?
Data niwe uri ku ngoma, Ijuru n' isi byishime.
